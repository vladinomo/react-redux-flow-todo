## これはなに

react+redux プロジェクトのボイラープレート。create-react-app で生成されたコードがベース

## 使い方

このレポジトリをローカルにクローンしてディレクトリに移動し`yarn install`

* `yarn start`で開発用サーバー起動`localhost:3000`でページにアクセス
* `yarn test`でテスト実行
* `yarn build`でプロダクション用ビルドを生成

## Stack

* React

* prettier
